Setup:

Step 1: Setup your development environment for PHP (for example WAMP - if you are using Windows)

Step 2: Copy the "SimplePHP_demo" folder to your web server’s root directory

Step 3: Enter your credentials in the edsapi-simple-app.php file (line 48-52)

Step 4: Start the demo (eg: http://localhost/SimplePHP_demo )

Code Basics

Authentication: By calling Authenticate method with UserID / Password (using HTTP POST)

Create Session: By calling CreateSession method with Profile and the x-authenticationToken HTTP header

Search: By calling Search method with the search criteria

Detailed record: By calling Retrieve method with the AN and Database ID
